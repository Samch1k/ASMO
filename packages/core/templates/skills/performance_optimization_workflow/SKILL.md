# Performance Optimization Workflow

## Overview

Data-driven performance optimization workflow ensuring measurable improvements through profiling, optimization, and validation.

## Process

1. **Analysis**: Profile application, identify bottlenecks
2. **Prioritization**: Rank optimizations by impact vs effort
3. **Implementation**: Apply optimizations (queries, caching, algorithms)
4. **Validation**: Measure improvements, ensure no regressions
