# Bug Fix Workflow

## Overview

Systematic approach to bug fixing ensuring root cause identification, proper fix implementation, and thorough verification.

## Process

1. **Diagnosis**: Analyze logs, stack traces, reproduce issue
2. **Reproduction**: Create minimal reproduction case
3. **Fix**: Implement hotfix with minimal changes
4. **Verification**: Test fix + regression testing
5. **Documentation**: Update bug tracker, add tests

- bug_diagnosis
- bug_reproduction
- hotfix_generation
- verification

## When to Use

- Fixing production bugs

- Handling customer-reported issues
