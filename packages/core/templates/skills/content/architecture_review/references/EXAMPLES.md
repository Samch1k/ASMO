# Architecture Review - Examples

## Example: Quarterly Architecture Review

**Scope**: Review MeatConnect platform architecture

**Process**:
1. Review 15 ADRs created in Q4 2025
2. Evaluate technology choices (PostgreSQL, Redis, React)
3. Assess scalability (current: 10K users, target: 100K)
4. Identify technical debt (3 critical items found)

**Output**:
- 2 new ADRs (caching strategy, database sharding plan)
- Technical debt prioritization list
- Recommendations for Q1 2026

**Time**: 8 hours
