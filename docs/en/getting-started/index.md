# Getting Started

Welcome to AI1st! This guide will help you get up and running with the AI1st orchestration framework.

## What is AI1st?

AI1st is an autonomous AI development orchestration system that coordinates multiple AI agents to work together as a cohesive team. Instead of treating AI as a simple assistant, AI1st transforms it into an autonomous development team capable of:

- **Multi-Agent Orchestration** - Coordinate 24 specialized agents
- **Adaptive Workflows** - Automatically select the right workflow for any task
- **Parallel Execution** - Run multiple agents simultaneously
- **Quality Gates** - Built-in approval checkpoints and validation

## Key Concepts Preview

Before diving in, here are the core concepts you'll encounter:

| Concept | Description |
|---------|-------------|
| **Agent** | A specialized AI role (e.g., Architect, Developer, Tester) |
| **Workflow** | A sequence of agent tasks to accomplish a goal |
| **Skill** | A specific capability that agents can use |
| **Complexity** | Task difficulty scoring (0-100) that drives workflow selection |
