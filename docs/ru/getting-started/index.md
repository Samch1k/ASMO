# Начало работы

Добро пожаловать в ASMO! Это руководство поможет вам начать работу с фреймворком ASMO.

## Что такое ASMO?

ASMO — это система автономной оркестрации AI-разработки, которая координирует работу нескольких AI-агентов как единой команды. Вместо использования AI как простого ассистента, ASMO превращает его в автономную команду разработки:

- **Мультиагентная оркестрация** — координация 24 специализированных агентов
- **Адаптивные workflows** — автоматический выбор подходящего workflow
- **Параллельное выполнение** — одновременная работа нескольких агентов
- **Контрольные точки** — встроенные checkpoints и валидация

## Ключевые концепции

| Концепция | Описание |
|-----------|----------|
| **Агент** | Специализированная AI-роль (Архитектор, Разработчик, Тестировщик) |
| **Workflow** | Последовательность задач агентов для достижения цели |
| **Навык** | Конкретная способность, которую агенты используют |
| **Сложность** | Оценка трудности задачи (0-100), определяющая выбор workflow |
