# Демонстрационный проект для ASMO

Это минимальный демонстрационный проект, используемый для демонстрации сценариев ASMO.

## Цель

Предоставляет реалистичные файлы для демонстрационных сценариев:
- Сценарий 1 (YOLO): Исправление опечатки в этом README
- Сценарий 2 (Каскад контекста): Добавление email валидации в форму регистрации

## Структура

```
sample-project/
  ├── README.md              (этот файл - для демонстрации исправления опечатки)
  ├── src/
  │   ├── components/
  │   │   └── SignupForm.tsx  (для демонстрации email валидации)
  │   └── api/
  │       └── validators/
  │           └── (email.ts будет создан ASMO)
  └── tests/
      └── (тестовые файлы будут созданы ASMO)
```

## Намеренная опечатка

Этот README содержит опечатку в этом предложении. ASMO должен найти и исправить её в Сценарии 1.
